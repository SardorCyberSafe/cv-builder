# 📄 CV Yaratuvchi — Resume Builder

Oddiy va tezkor CV yaratish vositasi. O'zbekiston foydalanuvchilari uchun maxsus ishlab chiqilgan.

**[🔗 Live Demo](https://sardorcybersafe.github.io/cv-builder)**

---

## ✨ Xususiyatlari

- **3 ta chiroyli shablon** — Oddiy, Zamonaviy, IT
- **Jonli preview** — Ma'lumot kirgizishingiz bilan darhol ko'rinadi
- **3 til qo'llab-quvvatlash** — 🇺🇿 O'zbek, 🇷🇺 Rus, 🇬🇧 Ingliz
- **PDF yuklab olish** — Bir tugma bilan tayyor CV
- **Avtomatik saqlash** — Ma'lumotlar brauzerda saqlanadi (localStorage)
- **Responsive dizayn** — Telefon, planshet va kompyuterda bir xil ishlaydi
- **Server kerak emas** — Faqat HTML/CSS/JS, statik hostingda ishlaydi

---

## 🚀 Ishga tushirish

### Oddiy usul

`index.html` faylini brauzerda oching — tayyor!

### Lokal server

```bash
# Python bilan
python3 -m http.server 8000

# Node.js bilan
npx serve .
```

## 🛠 Texnologiyalar

- **HTML5** — Semantik belgilash
- **CSS3** — CSS Grid, Flexbox, Custom Properties
- **Vanilla JavaScript** — Hech qanday framework kerak emas
- **[html2pdf.js](https://github.com/eKoopmans/html2pdf.js)** — PDF generatsiya
- **localStorage** — Ma'lumotlarni brauzerda saqlash

## 📝 Qanday ishlatiladi

1. Shaxsiy ma'lumotlaringizni kiriting
2. Ish tajribasi, ta'lim, ko'nikmalar, tillar va loyihalarni qo'shing
3. Sevimli shablonni tanlang
4. Jonli preview orqali natijani ko'ring
5. Tilni almashtiring (UZ / RU / EN)
6. **"PDF Yuklab Olish"** tugmasini bosing
